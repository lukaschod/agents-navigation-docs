    Struct UnsafeBHVTree<TVolume, TValue>

    An unmanaged, resizable bhv tree, without any thread safety check features. BHV tree (short for bounding hierarchy volume tree) is a space-partitioning data structure for organizing bounding shapes in space. As structure uses generic it is not only usable for boxes, but any shape that implements interfaces. Based on https://box2d.org/files/ErinCatto_DynamicBVH_GDC2019.pdf.
